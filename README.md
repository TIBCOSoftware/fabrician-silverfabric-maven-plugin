Silver Fabric Maven Plugin

## Help

`silverfabric:help`

#### Full name:

`com.tibco.silverfabric:silverfabric-maven-plugin:0.16:help`

#### Description:

Display help for the supported goals.

`mvn silverfabric:help -Ddetail=true -Dgoal=stacks`
